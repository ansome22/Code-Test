# Setup

## Run backend Locally

using pythons venv is recomended
python -m venv venv

OR

Docker

#### venv Windows machine

 - venv\Scripts\activate.bat
 - pip install -r backend/requirements.txt

#### venv mac/linux

source venv/bin/activate
pip install -r backend/requirements.txt


## Run front End
 - cd frontend
 - npm start
